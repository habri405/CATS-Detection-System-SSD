FAQs

What is CATS?

Conditional Analysis of Traffic Signs (CATS) is a web application for processing images and videos using object detection algorithms. It utilizes advanced machine learning models to recognize objects, including specific traffic signs.

How can I upload an image or video for object detection?

You can upload an image or video directly on the homepage. Click the "Choose File" button, select your file, and click "Upload" to start the detection process.

What types of files are supported for upload?

Currently, the application supports image formats like JPG, PNG and JPEG. For videos, common formats such as MP4 and AVI are accepted.

How do I view processed images or videos?

For images, a pop up of the image will automatically appear when the image is processed. For videos, a video player will pop up when the video is done processing. However, the player that pop ups cannot play said processed video. Instead the processed video can be viewed in the detections folder as well as the processed images.

What object detection models are used in CATS?

CATS system employs SSDMobileNetV2 model for object detection.

Is my data secure and private?

Yes, we prioritize user privacy and data security. We do not store uploaded files, and all processing is done on the server without retaining personal information.

How can I provide feedback or report issues?
